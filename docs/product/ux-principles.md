# UX Principles

## Experience Direction

- professional enough for astrologers to trust
- clear enough for non-experts to stay oriented
- spring-toned visual identity without becoming decorative noise

## Stable Interface Principles

- tables are for precise reading
- wheel visualizations are for spatial understanding
- Chinese terminology must be consistent and accurate
- names shown to the user should preserve the user input exactly

## Quality Bar

- tooltip wording must be precise
- chart labels, signs, aspects, and house references must feel native to the domain
- frontend polish matters because users will inspect small details closely

## Responsive And Accessibility Baseline

- core flows must work on desktop and mobile
- loading, error, and empty states should be explicit
- visual density should not make tables or wheels unreadable
