# Product Vision

Starry No Worry is growing toward a full-featured astrology charting platform. The current product goal is a professional-feeling astrology tool that users can trust for chart generation, chart reading, and structured interpretation.

## Current Phase

- active phase: Phase 3 bootstrap
- current focus: broader platform surfaces such as expanded location coverage, long-lived chart assets, export/share paths, and other post-Phase-2 product layers on top of the stable chart engine

## Durable Product Promise

- chart correctness comes before AI flourish
- advanced users should recognize the product as serious
- ordinary users should still find it readable and usable
