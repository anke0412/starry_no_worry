# Glossary

## Product Terms

- `本命盘`: natal chart
- `比较盘`: synastry chart
- `组合盘`: composite chart
- `Davison`: davison chart
- `流年盘`: transit chart
- `关系流年盘`: relationship transit chart
- `次限推运盘`: progression chart
- `日返盘`: solar return chart
- `月返盘`: lunar return chart

## Naming Rules

- use the user's entered name directly in presentation
- keep planet, sign, aspect, and house terminology professionally accurate
- do not invent casual synonyms for domain terms inside UI or reports

## Internal Framing

- `shared chart generation framework`: the reusable backend orchestration layer for chart families
- `Phase 2`: the current stage focused on advanced chart expansion, analysis interaction, and later AI interpretation enhancement
