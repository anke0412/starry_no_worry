# Operating Model

## Roles

### Human

- resolves product, roadmap, or risky architecture forks

### Main Agent

- restores context
- maintains the active plan
- chooses the current bounded step
- delegates and integrates results
- owns completion judgment

### Subagent

- executes bounded `impl`, `review`, or `verify` work
- returns concrete output and evidence
- does not redefine roadmap or stable truth
