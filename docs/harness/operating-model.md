# Operating Model

## Roles

### Human

- shapes the roadmap and resolves meaningful scope or architecture forks

### Main Agent

- reads stable docs first
- converts active work into one execution-ready plan
- delegates bounded work
- integrates outputs and owns completion judgment

### Subagent

- works on a narrow task slice
- returns concrete results and evidence
- does not redefine roadmap or architecture
